## 🤔 함수의 선언과 정의

함수는 먼저 선언되어야 하며, 선언된 함수는 필요한 위치에서 호출하여 사용할 수 있습니다.
함수의 선언은 함수의 이름, 반환 타입, 매개변수 목록으로 구성됩니다.
함수의 정의는 실제로 함수가 어떻게 동작하는지를 구현하는 부분을 의미합니다.

```c
// 함수의 선언
int add(int a, int b);

// 함수의 정의
int add(int a, int b) {
 return a + b;
}
```

위의 예제에서 add 함수는 두 개의 int 형 매개변수 a와 b를 받아서 더한 후 결과를 반환하는 함수입니다.
